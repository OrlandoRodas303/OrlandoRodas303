Guía para hacer pública tu página web y personalizar el URL

Pregunta:

Usuario: ¿Puedo editar el URL?

Respuesta:

ChatGPT: Sí, puedes personalizar el URL en varias plataformas dependiendo de cómo decidas alojar tu sitio web. A continuación te explico las opciones para editar el URL en las plataformas más comunes:

1. GitHub Pages

En GitHub Pages, el URL por defecto será algo como https://nombreusuario.github.io, pero puedes personalizar el URL siguiendo estas opciones:

Opción 1: Personalizar la URL del repositorio

Si creas un repositorio con un nombre diferente (que no sea nombreusuario.github.io), el URL será algo así como https://nombreusuario.github.io/nombreproyecto.

Opción 2: Usar un dominio personalizado

Puedes conectar un dominio que hayas comprado (como www.mipagina.com) a tu proyecto de GitHub Pages. Aquí están los pasos:

  1. Compra un dominio a través de proveedores como Namecheap o Google Domains.
  2. Configura el DNS de tu dominio apuntando a GitHub Pages con los registros A o CNAME:
    • IP de GitHub Pages: 185.199.108.153, 185.199.109.153, 185.199.110.153, 185.199.111.153
  3. Crea un archivo CNAME en tu repositorio con el nombre de dominio (ej: www.mipagina.com).

2. Netlify

En Netlify, puedes personalizar fácilmente la URL incluso si no compras un dominio:

Opción 1: Personalizar el subdominio gratuito

Netlify te asigna un subdominio como https://nombre-sitio.netlify.app. Puedes cambiar ese nombre en la configuración del sitio para que sea algo más personalizado, como mi-sitio-personalizado.netlify.app.

Opción 2: Usar un dominio personalizado

Al igual que con GitHub Pages, puedes conectar un dominio personalizado comprando uno y configurando el DNS en Netlify.

3. Hosting Tradicional

Si usas un servicio de hosting tradicional (como Hostinger, Bluehost), tienes la libertad de usar el dominio que quieras apuntando tu dominio al servidor del hosting.

Resumen:

  • GitHub Pages: Cambia el subdominio o conecta un dominio personalizado.
  • Netlify: Personaliza el subdominio o usa un dominio personalizado.
  • Hosting Tradicional: Total libertad para usar cualquier dominio que elijas.
